# Audio
> Digital audio in JavaScript.

A JavaScript object used to express LPCM digital audio.

## Installation
```shell
$ npm install --save audio
```

### Usage
```javascript
var Audio = require('audio');
```
See [the API docs (`API.md`)](/API.md) for more information.
